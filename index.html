🤖 Афиша недели через агента

Как это работает:
1️⃣ Выбираешь неделю → копируешь JSON событий
2️⃣ Открываешь агента в Claude.ai → вставляешь JSON → агент рисует 7 афиш
3️⃣ Скачиваешь PNG из агента → загружаешь сюда → постишь в Telegram в один клик
Шаг 1 · Выбрать неделю и выгрузить JSON
Шаг 2 · Загрузить готовые афиши (PNG)
Можно загрузить несколько PNG сразу — для каждого дня
Шаг 3 · Запостить в Telegram
К каждой PNG добавится подпись с датой и ссылками на запись. Каждая афиша = отдельный пост.
